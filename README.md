# Cypress E2E Saucedemo Framework

This is an end-to-end test automation framework built using Cypress and JavaScript.

## 📌 Features
- Page Object Model (POM)
- Cypress Fixtures for test data
- E2E checkout flow automation

## 🧪 Test Coverage
- Login
- Product selection
- Add to cart
- Checkout process

## 🚀 Tech Stack
- Cypress
- JavaScript

## ▶️ How to run

npm install
npx cypress open
